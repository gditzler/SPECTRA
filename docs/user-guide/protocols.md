# Aviation & Maritime Protocols

SPECTRA implements baseband IQ generators for several civilian protocol
waveforms used in aviation and maritime services. Each class produces
realistic frame-level signals with correct preamble, framing, and CRC,
suitable for protocol-aware detector training, recording-replay tests,
and synthetic dataset construction.

All protocol classes inherit `Waveform`, are deterministic given a seed,
and are safe across DataLoader workers.

## Available classes

| Class | Service | Modulation | Notes |
|-------|---------|------------|-------|
| `ADSB` | ADS-B 1090 MHz Extended Squitter | PPM | 112-bit frames, 8 µs preamble |
| `ModeS` | Mode S Interrogation/Reply | PPM | 56- or 112-bit messages, 8 µs preamble |
| `AIS` | Automatic Identification System | GMSK (BT=0.4, 9600 baud) | Maritime vessel tracking |
| `ACARS` | Aircraft Communications Addressing and Reporting System | AM-MSK (2400 baud) | Air-ground data link |
| `DME` | Distance Measuring Equipment | Gaussian pulse pairs | Pulsed; configurable pulse width and spacing |
| `ILS_Localizer` | ILS Localizer | AM (90 Hz + 150 Hz tones) | DDM encodes lateral deviation |

## Quickstart

```python
from spectra.waveforms.aviation_maritime import ADSB

wf = ADSB()
iq = wf.generate(num_symbols=16, sample_rate=2e6, seed=0)
print(wf.label, iq.shape)
# ADSB (38400,)
```

## ADSB — 1090 MHz Extended Squitter

ADS-B generates PPM-encoded signals with a standard 8 µs preamble
(pulses at 0, 1, 3.5, 4.5 µs) followed by 112 data bits. Each bit
period is 1 µs; a `1` encodes as `[1, 0]` and a `0` as `[0, 1]` on
half-bit (0.5 µs) chips. Frame bytes are generated by the Rust
`generate_adsb_frame` function with per-frame seeding.

**Constructor**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `samples_per_chip` | `int` | `10` | Samples per 0.5 µs chip period |

```python
from spectra.waveforms.aviation_maritime import ADSB

adsb = ADSB(samples_per_chip=10)
iq = adsb.generate(num_symbols=8, sample_rate=2e6, seed=42)
print(adsb.label, iq.shape)
# ADSB (19200,)
```

`bandwidth()` returns `sample_rate / samples_per_chip`.

---

## ModeS — Mode S Interrogation/Reply

Mode S shares the same preamble structure and PPM encoding as ADS-B but
supports configurable message lengths: 56-bit short squitters and
112-bit long squitters. Only the values `56` and `112` are accepted;
any other value raises `ValueError`.

**Constructor**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `message_length` | `int` | `112` | Number of data bits: `56` or `112` |
| `samples_per_chip` | `int` | `10` | Samples per 0.5 µs chip period |

```python
from spectra.waveforms.aviation_maritime import ModeS

# 56-bit short squitter
modes_short = ModeS(message_length=56, samples_per_chip=10)
iq = modes_short.generate(num_symbols=8, sample_rate=2e6, seed=42)
print(modes_short.label, iq.shape)
# ModeS (10240,)

# 112-bit long squitter (default)
modes_long = ModeS(message_length=112)
iq = modes_long.generate(num_symbols=8, sample_rate=2e6, seed=42)
print(modes_long.label, iq.shape)
# ModeS (19200,)
```

`bandwidth()` returns `sample_rate / samples_per_chip`.

## AIS — Automatic Identification System

AIS uses GMSK modulation with BT=0.4 at 9600 baud for maritime vessel
identification and tracking. The generator applies NRZ encoding, upsamples
with zero-insertion, filters with a Gaussian pulse, and integrates phase
for continuous-phase MSK output (modulation index 0.5).

**Constructor**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `samples_per_symbol` | `int` | `8` | Samples per symbol period |
| `bt` | `float` | `0.4` | Gaussian filter bandwidth-time product |
| `filter_span` | `int` | `4` | Gaussian filter span in symbols |

```python
from spectra.waveforms.aviation_maritime import AIS

ais = AIS(samples_per_symbol=8, bt=0.4, filter_span=4)
iq = ais.generate(num_symbols=8, sample_rate=2e6, seed=42)
print(ais.label, iq.shape)
# AIS (13312,)
```

`bandwidth()` returns `(sample_rate / samples_per_symbol) * (1.0 + bt)`.
At the defaults with `sample_rate=2e6` this is 350 kHz.

---

## ACARS — Aircraft Communications Addressing and Reporting System

ACARS uses AM-MSK modulation at 2400 baud for air-to-ground data
communications. Internally, the generator applies NRZ encoding, upsamples,
then integrates phase increments to produce continuous-phase FSK output
with modulation index 0.5.

**Constructor**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `samples_per_symbol` | `int` | `8` | Samples per symbol period |

```python
from spectra.waveforms.aviation_maritime import ACARS

acars = ACARS(samples_per_symbol=8)
iq = acars.generate(num_symbols=8, sample_rate=2e6, seed=42)
print(acars.label, iq.shape)
# ACARS (14912,)
```

`bandwidth()` returns `(sample_rate / samples_per_symbol) * 1.5`.
At the defaults with `sample_rate=2e6` this is 375 kHz.

## DME — Distance Measuring Equipment

DME generates Gaussian pulse pairs with configurable pulse width and
inter-pulse spacing. Each call to `generate()` produces `num_symbols`
pulse pairs separated by random inter-pair gaps drawn uniformly from
20–100 µs.

**Constructor**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `pulse_width_us` | `float` | `3.5` | Gaussian pulse width in microseconds (FWHM) |
| `pulse_spacing_us` | `float` | `12.0` | Spacing between pulses in a pair (µs) |
| `samples_per_us` | `int` | `10` | Number of samples per microsecond |

```python
from spectra.waveforms.aviation_maritime import DME

dme = DME(pulse_width_us=3.5, pulse_spacing_us=12.0, samples_per_us=10)
iq = dme.generate(num_symbols=8, sample_rate=2e6, seed=42)
print(dme.label, iq.shape)
# DME (7576,)
```

`bandwidth()` approximates Gaussian pulse bandwidth as
`1 / (2π × σ)` where `σ` is derived from the FWHM, capped at
`sample_rate`. At defaults this is roughly 107 kHz.

Note: because the output length depends on random inter-pair gaps, the
exact shape will vary across seeds even for the same `num_symbols`.

## ILS Localizer

The ILS Localizer represents an AM carrier modulated by 90 Hz and
150 Hz tones. The Difference in Depth of Modulation (DDM) between
the two tones encodes lateral deviation from the runway centerline.
A small random DDM offset is added each call for realism.

**Constructor**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `modulation_depth` | `float` | `0.2` | Depth of AM modulation (0 to 1) |
| `samples_per_symbol` | `int` | `256` | Samples per symbol; controls output length |

```python
from spectra.waveforms.aviation_maritime import ILS_Localizer

ils = ILS_Localizer(modulation_depth=0.2, samples_per_symbol=256)
iq = ils.generate(num_symbols=8, sample_rate=2e6, seed=42)
print(ils.label, iq.shape)
# ILS_Localizer (2048,)
```

`bandwidth()` returns `min(300.0, sample_rate)` — the localizer is very
narrowband (tones at 90 and 150 Hz).

## Use in a scene

Protocol waveforms compose directly with `Composer` for multi-signal
wideband scene generation. The scene automatically frequency-shifts each
signal within the capture bandwidth. Classes with narrow bandwidth (e.g.
`ILS_Localizer` at ~300 Hz) will occupy very little spectral space;
broadband capture bandwidths are preferable for multi-signal scenes.

```python
from spectra.scene import Composer, SceneConfig
from spectra.waveforms.aviation_maritime import AIS, ADSB

scene = Composer(SceneConfig(
    capture_duration=10e-3,
    capture_bandwidth=2e6,
    sample_rate=2e6,
    num_signals=2,
    signal_pool=[AIS(), ADSB()],
    snr_range=(5.0, 20.0),
))
iq, descs = scene.generate(seed=0)
print(iq.shape, len(descs))
for d in descs:
    print(d.label, d.f_low / 1e3, "to", d.f_high / 1e3, "kHz  SNR:", round(d.snr, 1), "dB")
```

## Use in a NarrowbandDataset

```python
from torch.utils.data import DataLoader
from spectra.datasets import NarrowbandDataset
from spectra.waveforms.aviation_maritime import ADSB, ModeS, AIS, ACARS, DME, ILS_Localizer

ds = NarrowbandDataset(
    waveform_pool=[ADSB(), ModeS(), AIS(), ACARS(), DME(), ILS_Localizer()],
    num_samples=600,
    num_iq_samples=512,
    sample_rate=2e6,
    seed=0,
)
loader = DataLoader(ds, batch_size=32, num_workers=2)
iq_batch, labels = next(iter(loader))
print(iq_batch.shape, labels.shape)
# torch.Size([32, 2, 512]) torch.Size([32])
```
